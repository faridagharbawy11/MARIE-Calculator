# MARIE Assembly Calculator

This project implements a basic calculator using MARIE Assembly language. It supports the four fundamental arithmetic operations:

- **Addition** (`+`)
- **Subtraction** (`-`)
- **Multiplication** (`*`)
- **Division** (`/`)
